# Changelogs

### December 4, 2025

- **Breadcrumb customization**: Added an option to hide products name from breadcrumbs and improved mobile styling with new customization options for font size and spacing using custom CSS code.

### December 1, 2025

- **Product category filter**: You can now filter products based on whether their category is empty or not empty.

### November 12, 2025

- **Breadcrumb settings**: Added a new settings page to define priority for breadcrumbs and customize how they appear.
- **Collection breadcrumb customization**: You can now change the root label for collection breadcrumbs (defaults to "home").
- **Product breadcrumb app block**: Added a new app block for displaying product breadcrumbs in your theme.
- **New filter**: Added a new **media count** filter to select products based on the number of images or media files they have.

### October 26, 2025

- **Import improvements**: Enhanced support for importing collections with improved feedback and error messages.

### September 19, 2025

- **Duplicate collection branch**: New tool to duplicate a full collection branch (early access)

### August 30, 2025

- **New filter**: Added an **inventory tracking** filter to select products based on whether inventory tracking is enabled or disabled.

### August 21, 2025

- **Nested collections fix**: Fixed a bug that prevented parent collections from being properly processed for products at depth 3 or deeper in the hierarchy.

### August 15, 2025

- **Collection grid improvements**: Enhanced the collection grid app block with more settings.

### August 13, 2025

- **Product with over 100 variants**: Fixed processing for products having more than 100 variants (extra variants were skipped).

### August 6, 2025

- **Settings page fix**: Fixed an issue that prevented saving settings.
- **Parent collection picker fix**: Fixed the parent collection picker to properly show a modal when no collections are available.
- **Increased collection limits**: Raised the maximum number of collections allowed for each plan.

### July 21, 2025

- **New date operator**: Added a new operator **"is more than X days ago"** for date filters, making it easier to filter products by relative dates.
- **Empty/not empty for dates**: The "is empty" and "is not empty" operators are now available for date metafields and product publish dates.

### June 30, 2025

- **Root collection metafields**: Added new metafields for storing up to 4 root collections, with updated theme extension support for better collection hierarchy management.
- **Date filter fix**: Fixed the "relative last days" filter to correctly exclude future dates.

### June 26, 2025

- **Duplicate collections**: You can now duplicate existing managed collections, making it faster to create similar collections with pre-filled conditions.

### June 24, 2025

- **Import/Export tool**: Added a new import and export tool for managed collections (early access).

### June 21, 2025

- **Date and datetime metafield support**: Added support for filtering on **date** and **date_time** metafield types.
- **Swedish language**: The app is now available in Swedish.
- **Discount comparison operators**: Added new operators to filter discounts by percentage - **"less than X%"** and **"greater than X%"**.

### June 20, 2025

- **Collection creation improvements**: Fixed redirect issues after creating collections and added a helpful banner suggesting you create another managed collection.
- **Better keyboard navigation**: Improved keyboard interaction for condition groups and filters.

### June 19, 2025

- **Settings page**: Added a new settings page where you can customize the app's homepage and choose which page you see when you first open the app.

### June 18, 2025

- **Warning for missing collection**: If a collection is manually deleted from Shopify and was previously linked to a managed collection in the app, a warning message will now appear on the app's homepage. This helps you identify and fix managed collections that no longer have an associated Shopify collection.

### June 16, 2025

- **Search collections**: We’ve added a text field to the managed collections list so you can search for the collection you’re looking for.

### June 8, 2025

- **New metafield support**: We've added support for metafields that store **metaobject** and **list of metaobjects**. They can be used to filters products that have a given metaobject.

### June 5, 2025

- **Fixed tag filtering**: When you use "is not equal" or "does not contain" for tags, the condition now correctly skips any product with at least one tag matching the condition.
- **Renamed and added stock filters**: The old **Inventory stock** filter is now called **Total inventory stock**, and there’s a new **Variant inventory stock** filter so you can choose products based on the stock of each variant.

### June 3, 2025

- **Improved metafields search**: When selecting a condition (product field or metafield), the list is now searchable. Additionally, product metafields and variant metafields are divided into two sections for easier navigation.

### June 2, 2025

- **New filtering operators**: In addition to the existing operators such as "equal," "greater than," etc., we've added new operators **"is empty"** and **"is not empty"**. They have been added for all **metafields** and the product field **"compare-at price"**, allowing filtering of blank data.
- **New filter**: We've added a new filter named **"discount"**. It allows filtering product that **are** or **are not** on discount.

### May 30, 2025

- **Numerous performance improvments**: Smart Collection Pro is now more stable for larger stores.

### May 26, 2025

- **New metafield support**: We've added support for metafields that store **booleans**.

### April 23, 2025

- **New translations available**: The app is now available in German, Dutch and Turkish.

### April 21, 2025

- **Better support for special characters**: We've improved how the app handles characters from non latin scripts and special characters like French accents, the German “ß” (now matched with “ss”), and Turkish letters that use combined characters (like accented “g”), Asian scripts. This means searches and filters will behave more naturally across languages, including French, Czech, German, Turkish, Greek, Japanase, Korean, Chinese...

### April 18, 2025

- **Filter by product dates**: You can now filter products based on their **creation or publish date**, making it possible to build, for example, collections that list products released in the last 30 days.

### April 15, 2025

- **New metafield support**: We've added support for metafields that store **decimals** and **integers**.

### April 3, 2025

- **Smarter tag selection**: Tag fields now suggest options as you type, so adding tags is faster and easier.

### April 2, 2025

- **New metafield support**: We've added support for metafields that store **lists of single-line text**.

### March 27, 2025

- **New translations available**: The app is now available in Czech, French, Italian, Spanish, Japanese, Korean, Danish, and Portuguese.
